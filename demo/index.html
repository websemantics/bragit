Show your Github repositories stats in style using Semantic UI labeled buttons.

How To

1- Install Semantic-ui and Semanti library using Bower. This will also install jQuery.

Bower install semantic-ui Bower install semanti

2- Include the required Javascript / CSS files in the HTML page and initilize Semanti with Github username and repository name.

<link href="bower_components/semantic-ui/dist/semantic.min.css" type="text/css" rel="stylesheet"> <script src="bower_components/jquery/dist/jquery.min.js" type="text/javascript"></script> <script src="bower_components/semanti/semanti.js" type="text/javascript"></script> <script type="text/javascript"> $(function() { Semanti.init(/* Github username */ 'websemantics', /* Github repo */ 'semanti'); }); </script>

3- Use Semantic-ui labeled buttons anywhere in your page. Here's an example of the famous Github stars button.
