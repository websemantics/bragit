Github

Show off your Github repository buttons with elegance.

What is it

An eforetless way to present Github stats buttons of your repositories while not compromise on style using the amazing Semantic-ui framework.

How To

1- Include the following in your page.

<script type="text/javascript" src="https://cdn.rawgit.com/websemantics/bragit/0.1.2/bragit.js"></script>

2- Insert a labled button as specified in Semantic-ui documentation and add a class name for the Github button you desire. For example, to show the stars buttons of Bragit repository the class would be github-websemantics-bragit-stars
